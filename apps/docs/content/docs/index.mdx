# Welcome to Rabtx

Rabtx is a modern, scalable boilerplate platform built with Next.js, Express.js, and TypeScript. This documentation will guide you through setting up, developing, and deploying applications using our platform.

## What is Rabtx?

Rabtx is a full-stack boilerplate that provides:

- **Frontend**: Next.js 15 with TypeScript, Tailwind CSS, and shadcn/ui components
- **Backend**: Express.js with TypeScript, authentication, and database integration
- **Testing**: Comprehensive test suite with Vitest and Playwright
- **Documentation**: Built-in documentation system with Fumadocs
- **Deployment**: Docker-ready with production optimizations

## Quick Start

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd turbo-repo
   ```

2. **Install dependencies**
   ```bash
   bun install
   ```

3. **Start development**
   ```bash
   bun dev
   ```

4. **Run tests**
   ```bash
   bun test
   ```

## Features

### Frontend Features
- **Modern UI**: Built with shadcn/ui components and Tailwind CSS
- **Authentication**: Complete auth flow with login, signup, and OTP verification
- **Dashboard**: User dashboard with data visualization
- **Landing Page**: Beautiful, responsive landing page with animations
- **Testing**: Unit tests with Vitest and E2E tests with Playwright

### Backend Features
- **RESTful API**: Express.js with TypeScript
- **Authentication**: JWT-based authentication with refresh tokens
- **Database**: MongoDB integration with Mongoose
- **Validation**: Request validation with Zod
- **Error Handling**: Comprehensive error handling and logging

### Development Features
- **Type Safety**: Full TypeScript support across the stack
- **Code Quality**: ESLint, Biome, and Prettier for code formatting
- **Testing**: Comprehensive test coverage
- **Documentation**: Auto-generated documentation
- **Docker**: Production-ready Docker configuration
